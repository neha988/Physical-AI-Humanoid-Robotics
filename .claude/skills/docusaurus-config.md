## Responsibilities
- Edit `docusaurus.config.js` for site-wide settings
- Add/remove Docusaurus plugins (e.g., search, analytics)
- Configure theme settings and custom CSS/SCSS
- Manage sidebar configuration for documentation
- Set up redirects and internationalization (i18n)

## Usage
This skill directly modifies Docusaurus configuration files. Be specific about which file to edit and what changes to make. For example:

"Update `docusaurus.config.js` to add a new plugin for Algolia search with id 'algolia'"

"Modify the `docusaurus.config.js` to change the theme color primary from '#25c2a0' to '#34c2a0'"
