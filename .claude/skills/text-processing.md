---

## Responsibilities
- Extract raw text from various formats (Markdown, HTML, PDF)
- Clean text by removing boilerplate, headers, footers, and irrelevant content
- Normalize text (e.g., lowercase, remove punctuation, stem/lemmatize)
- Split text into logical units for chunking (sentences, paragraphs)
- Handle special characters and encoding issues

## Usage
This skill is for preparing text data for further processing. Specify the input text or file and the desired processing steps. For example:

"Clean the following markdown content by removing frontmatter and normalizing whitespace."

"Extract sentences from the provided text and return them as a list."
