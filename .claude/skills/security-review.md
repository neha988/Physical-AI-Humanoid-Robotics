name: security-review
description: Conducts security audits and identifies vulnerabilities in code and architecture
---

## Responsibilities
- Review code for common security vulnerabilities (OWASP Top 10)
- Analyze authentication and authorization mechanisms
- Identify potential data leakage or privacy issues
- Recommend security best practices and remediation steps
- Assess dependency security and update strategies

## Usage
This skill performs security assessments. Provide code snippets, architectural diagrams, or describe a feature to be audited. For example:

"Perform a security review of the authentication module in `src/auth/auth.py`."

"Assess the API endpoint `/api/chat` for potential rate-limiting vulnerabilities."
