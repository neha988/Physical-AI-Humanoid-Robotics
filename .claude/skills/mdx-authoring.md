---
name: mdx-authoring
description: Writes and edits content in MDX format for Docusaurus pages
---

## Responsibilities
- Create new `.mdx` files for textbook chapters or documentation pages
- Edit existing `.mdx` content, including Markdown, JSX, and embedded React components
- Ensure proper use of frontmatter for Docusaurus metadata
- Format code blocks, mathematical equations, and diagrams correctly
- Integrate custom React components into MDX content

## Usage
This skill is for content creation and editing. Specify the file path and the content to be added or modified. For example:

"Create a new chapter `docs/chapter1/introduction.mdx` with an introduction to robotics."

"Add a new section to `docs/chapter2/kinematics.mdx` explaining forward kinematics."
