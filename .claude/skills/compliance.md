---
name: compliance
description: Ensures adherence to security standards, regulations, and internal policies
---

## Responsibilities
- Review system against compliance requirements (e.g., GDPR, HIPAA, ISO 27001)
- Advise on data privacy and retention policies
- Ensure logging and auditing mechanisms meet standards
- Help implement security controls to achieve compliance
- Document compliance posture and gaps

## Usage
This skill provides guidance on compliance. Specify the standard or regulation and the system/feature to be assessed. For example:

"Assess the user data handling in the backend API for GDPR compliance."

"Ensure that password policies meet internal security compliance standards."
