## Responsibilities
- Create `Dockerfile` for containerizing frontend or backend applications
- Optimize Docker images for size and build speed
- Manage Docker Compose configurations for multi-service applications
- Integrate Docker builds into CI/CD pipelines
- Troubleshoot containerization issues

## Usage
This skill is for Docker-related tasks. Specify the application to containerize and any specific requirements. For example:

"Create a Dockerfile for the FastAPI backend application."

"Optimize the existing Dockerfile to reduce image size."
